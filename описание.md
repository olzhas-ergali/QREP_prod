**QREP_qa** — тестовое окружение бота и API

**Назначение:** QA-версия бота и отдельный HTTP API для тестирования.

- **Стек:** Python, aiogram, FastAPI (uvicorn), PostgreSQL, Redis
- **Состав:**
  - **Бот** — тот же функционал, что и в prod (staff, client, probation, покупки и т.д.), но для тестов
  - **API** — веб-сервис на порту **8445** (`api.py` → `service.API.main:app`)
- **Запуск:**
  - Бот: `python bot.py`
  - API: `python api.py` (uvicorn на `0.0.0.0:8445`)
